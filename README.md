# Customer Complaint Root-Cause Analyzer (RAG)

This project uses Retrieval-Augmented Generation (RAG) to analyze customer complaints
and identify recurring issues and product-impact changes after releases.

## Data Sources
- Customer Complaint Records
- Customer Support Resolution Notes
- Product Release Notes

## Tech Stack
- Python
- LangChain
- FAISS
- Sentence Transformers
- Mistral (via Ollama)
- Streamlit / FastAPI
